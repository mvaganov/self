## ARModelPlatform
AR model viewer for Android. Load and view gltf models in aug.ented reality. First in Unity, then in Kotlin, then in C, then compare pros and cons of Unity/Kotlin/C, then use all 3 for different purposes in the same app (Unity for UI, Kotlin for business logic, C for rendering and bulk processing).

## UProblems
LLM that searches for problems in the world using web scraping (social media and prediction markets are rich data sources), and ranks those problems on a variety of metrics (commonality, severity, expense, ...). Publish that data, offer sources for a fee, or to subscribers. Offer to hide entries to the free tier viewers for a monthly subscription. Example: List the richest people in the world. Provide evidence, scraped from the web. Offer to hide names from public scrutiny for a minthly subscription fee relative to total wealth, never exceeding the cost of a hackers' weekly rate.

## DecisionTools
Create basic portable APIs (using pure text interfaces) for several decision making technologies: Djikstra's algorithm, A* algorithm, Ranked Choice Voting, Alpha Beta Pruning, Nash Equilibrium solver, ... . Should be testable in a command line interface.

## UniCalc
Create a spreadsheet app in Unity. Use actual JavaScript as the scripting language instead of the terrible function APIs. Use the actual V8 VM if possible, or a minimal C# port. Make an interface that can arrange thousands of different spreadsheets in a 3d environment. Ultimate target is VR/AR.

## Project Management Game
Mobile game about project management. Create user interface of Work Breakdown Structures, Gantt charts, Burndown Charts, Kanban boards, spreadsheets. The game should be an agent based model where NPCs accomplish complex quest lines. The project management tools provide visibility on their progress.

## VRBuilder
VR Game about building real homes using real construction techniques. Simple 3D model creation and manipulation -> terrain deformation -> home infrastructure modeling -> electrical and fluid dynamic modeling -> out door weather modeling

## VisInventory
Location aware camera app that recognizes stuff, and catalogs where that stuff was last seen.

## Habit tracker
People identify a number in their life, and an expected rate of change. The app should calculate those expectations over time and contact individuals regularly to determine accuracy of expectations, seeking info about cause/effect to generate better estimations.

## Collectivism
Create a sharing economy app that allows people to pool their resources together for economic benefits (tax evasion, economy of scale from bulk ordering, arbitrage opportunities from communal labor). Each member should have entries for skills and resources that can be shared, and personal goals/needs. AI defines tasks for maximum collective benefit.

## RhymeAReason
LLM that can rhyme a desired message concisely. Ultimate goal is to create very catchy political messaging to change opinions using the power of the Rhyme As Reason cognitive bias.
