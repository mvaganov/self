## ARModelPlatform
AR model viewer for Android. Load and view gltf models in augmented reality. First buildnit in Unity, then in Kotlin, then in C, then compare pros and cons of Unity/Kotlin/C, then use all 3 for different purposes in the same app (Unity for UI, Kotlin for business logic, C for rendering and bulk processing). the ultimate target for this is AR glasses, which are hyper constrained devices, and need expertise with efficiency and architecture level optimization.

## UProblems
LLM that searches for problems in the world using web scraping (social media and prediction markets are rich data sources), and ranks those problems on a variety of metrics (commonality, severity, expense, ...). Publish that data, offer sources for a fee, or to subscribers. Offer to hide entries to the free tier viewers for a monthly subscription. Subscription fee should never exceeding the cost of a hackers' weekly rate.

## DecisionTools
Create basic portable APIs (using pure text/serial interfaces) for several decision making technologies: Djikstra's algorithm, A* algorithm, Ranked Choice Voting, Alpha Beta Pruning, Nash Equilibrium solver, ... . Should be usable (and of course testable) in a pure command line interface. LLM agents will want to use these tools.

## UniCalc
Create a spreadsheet app in Unity. Use actual JavaScript as the scripting language instead of the terrible function APIs given by MS Excel. Use the actual V8 VM if possible, or a minimal C# port. Make an interface that can arrange thousands of different spreadsheets in a 3d environment. Ultimate target is VR/AR. Well organized data that shows up at the location it is needed seems really useful.

## Project Management Game
(mobile) Game about project management. Create user interface of Work Breakdown Structures, Gantt charts, Burndown Charts, Kanban boards, spreadsheets. The game should be an agent based model where NPCs accomplish complex quest lines. The project management tools provide visibility on their progress. build a fun interface for project management to include more people in big projects.

## VRBuilder
VR Game about building real homes using real construction techniques. Simple 3D model creation and manipulation -> terrain deformation -> home infrastructure modeling -> electrical and fluid dynamic modeling -> out door weather modeling. general contracting suffers from a lack of skill, lack of workers, and huge cost overruns that could be averted with better planning and simulation. this app seems like it would address all of those problems.

## Habit tracker
People identify a number in their life, and an expected rate of change. The app should calculate those expectations over time and contact individuals regularly to determine accuracy of expectations, seeking info about cause/effect to generate better estimations. if the app can be connected to sensing systems, it should be able to verify habits.

## Collectivism
Create a sharing economy app that allows people to pool their resources together for economic benefits (tax evasion, economy of scale from bulk ordering, arbitrage opportunities from communal labor). Each member should have entries for skills and resources that can be shared, and personal goals/needs. AI defines tasks for maximum collective benefit. this is actually a huge piece of software that automates management, from CEOs down to most of middle management. getting this right will flip the power structure in capitalism from owners of capital to workers, since capital ownership should theoretically be shared and managed by a low-ambition stakeholder (the AI).

## RhymeAsReason
LLM that can rhyme a desired message concisely. Ultimate goal is to create very catchy messaging to change opinions using the power of the Rhyme As Reason cognitive bias. possibly already being created with Sonu.
